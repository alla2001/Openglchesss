OpenGLChess is a graphics renderer I built for learning purposes. Currently, it supports:

-  OBJ file format for meshes
-  Forward rendering
-  Point lights
-  Shadow maps and soft shadow rendering
-  Skyboxes
-  Image-based lighting
-  PBR (Physically Based Rendering)
